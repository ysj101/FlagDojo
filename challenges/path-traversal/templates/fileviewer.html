What is Path Traversal?

Path traversal (also known as directory traversal) is a web security vulnerability that allows an attacker to read arbitrary files on the server.

By manipulating file paths with sequences like ../, attackers can escape from the intended directory and access sensitive files.

Prevention:

  • Validate and sanitize all file path inputs
  • Use allowlists of permitted files
  • Resolve the full path and check it's within the allowed directory
  • Use built-in framework functions for safe file access
